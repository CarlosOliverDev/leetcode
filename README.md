# LeetCode - Java

Este repositório contém minha jornada de resolução de problemas no **LeetCode**, focado em aprimorar conhecimentos sobre **Algoritmos** e **Estruturas de Dados**.

Todas as soluções são implementadas em **Java** e incluem casos de teste locais(`main method`) para validação imediata da lógica.

## Objetivo
Aplicar conceitos de Engenharia de Computação na resolução de problemas complexos, focando em:
- Análise de complexidade(Big O Notation).
- Manipulação eficiente de estruturas de dados(Arrays, Maps, Lists).
- Código limpo e legível.

## Estrutura do Repositório
Cada arquivo `.java` representa um problema único e contém:
1.  **Enunciado/Link:** Referência direta ao problema original.
2.  **Solução:** A implementação do algoritmo.
3.  **Teste Local:** Um método `main` com exemplos de entrada e saída para rodar o código sem depender da plataforma.
